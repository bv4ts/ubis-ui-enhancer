## ğŸ› ï¸ Installation

### Official Version (Recommended)
You can install the stable version directly from the **[Chrome Web Store](https://chromewebstore.google.com/detail/icmckbjcapeekdnomolkhejaenbbpnnd?utm_source=item-share-cb)**.

### Developer Version (Local Install)
1.  Clone this repository or download the source code.
2.  Open Chrome and navigate to `chrome://extensions/`.
3.  Enable **"Developer mode"** (top right toggle).
4.  Click **"Load unpacked"** and select the folder containing the project files.
